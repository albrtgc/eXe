4.1 Fundamentos y creación de bases de datos

En este bloque comenzarás a trabajar directamente con MySQL, aprendiendo a crear bases de datos, tablas y registros mediante instrucciones SQL. Es el primer paso para construir un modelo de datos completo y coherente.

Conceptos clave

  • Base de datos (BD): conjunto estructurado de información almacenada de forma organizada.
  • Tabla: estructura que almacena registros con campos o columnas definidas.
  • Registro: fila dentro de una tabla, representa una instancia concreta.
  • Campo: columna de una tabla que contiene un tipo de dato específico.
  • Clave primaria (PK): identifica de forma única cada registro de una tabla.

Creación de una base de datos

Para crear una base de datos en MySQL se utiliza la instrucción CREATE DATABASE:

CREATE DATABASE tienda
CHARACTER SET utf8mb4
COLLATE utf8mb4_general_ci;

La codificación utf8mb4 garantiza compatibilidad con caracteres multilingües y emojis.

Selección y uso de una base de datos

USE tienda;

Este comando establece la base de datos activa para las operaciones siguientes.

Creación de tablas

CREATE TABLE productos (
  id INT AUTO_INCREMENT PRIMARY KEY,
  nombre VARCHAR(100) NOT NULL,
  precio DECIMAL(10,2) NOT NULL DEFAULT 0.00,
  stock INT NOT NULL DEFAULT 0,
  fecha_alta DATE
);

Las cláusulas NOT NULL y DEFAULT establecen restricciones para garantizar la validez de los datos. AUTO_INCREMENT genera automáticamente valores consecutivos para la columna id.

Modificación de tablas

ALTER TABLE productos
ADD COLUMN descripcion TEXT AFTER nombre;

También puedes eliminar una columna:

ALTER TABLE productos DROP COLUMN descripcion;

Inserción de datos

INSERT INTO productos (nombre, precio, stock, fecha_alta)
VALUES ('Teclado mecánico', 79.95, 10, '2024-01-10'),
       ('Ratón inalámbrico', 24.50, 25, '2024-01-11');

Para visualizar los registros insertados:

SELECT * FROM productos;

Eliminación de registros

DELETE FROM productos WHERE id = 2;

Actividad práctica guiada

  1. Crea una base de datos llamada empresa.
  2. Dentro de ella, crea una tabla empleados con campos: id, nombre, puesto, salario y fecha_alta.
  3. Inserta tres registros de ejemplo.
  4. Realiza una consulta que muestre solo los empleados cuyo salario supere los 1500 €.
