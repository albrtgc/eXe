2.4 Arquitectura servidor–cliente

MySQL sigue un modelo de arquitectura servidor–cliente:

  • El servidor MySQL es el proceso que gestiona las bases de datos y atiende las peticiones de los usuarios.
  • Los clientes (como la consola mysql, phpMyAdmin o MySQL Workbench) se conectan al servidor mediante un protocolo de red.

Las conexiones requieren usuario, contraseña y, en algunos casos, permisos específicos. Es una buena práctica crear un usuario diferente a root para las aplicaciones.
