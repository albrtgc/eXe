2. Bienvenida al entorno MySQL

En esta segunda sección del REA “MySQL paso a paso: fundamentos, consultas y relaciones”, el alumnado se introduce en el entorno del sistema gestor de bases de datos MySQL. Este bloque proporciona una base conceptual sólida para comprender qué es un SGBD relacional, cómo se integra MySQL en el ecosistema de desarrollo web y de qué manera se estructura internamente.

Antes de comenzar a instalar y practicar, es importante conocer por qué MySQL es tan utilizado y cuáles son sus componentes esenciales. Aquí aprenderás las diferencias entre MySQL, MariaDB y PostgreSQL, así como la forma en que se comunican los clientes con el servidor de bases de datos mediante el modelo cliente–servidor.

Objetivos del bloque

  • Comprender el papel de MySQL dentro del ecosistema de desarrollo web.
  • Distinguir las características de los principales sistemas gestores de bases de datos.
  • Reconocer la arquitectura servidor–cliente y los elementos que la componen.
  • Familiarizarse con la terminología básica: tablas, registros, claves, relaciones y consultas.

Contenidos que se desarrollan

  1. 2.1 ¿Qué es un SGBD relacional?: fundamentos de la gestión de datos estructurados.
  2. 2.2 MySQL en el ecosistema LAMP: relación entre Linux, Apache, MySQL y PHP.
  3. 2.3 MySQL vs MariaDB vs PostgreSQL: comparación entre sistemas gestores.
  4. 2.4 Arquitectura servidor–cliente: estructura interna y comunicación entre componentes.
  5. 2.5 Actividades interactivas: cuestionarios, emparejamientos y reflexiones guiadas.

Metodología

La sección combina contenidos teóricos breves con actividades interactivas que promueven la comprensión de los conceptos básicos de MySQL. Se recomienda dedicar aproximadamente 1,5 horas a este bloque, asegurando que el alumnado asimile la terminología y el funcionamiento general antes de pasar a la instalación.

Actividad inicial de motivación: Piensa en una aplicación web que utilices a diario (por ejemplo, un comercio electrónico o una red social). ¿Qué tipo de datos crees que almacena en su base de datos MySQL?

Duración estimada: 1,5 horas · Tipo de contenido: Teórico-introductorio · Nivel: Inicial DAW
