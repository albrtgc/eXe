3.1 Instalación mediante XAMPP / Docker

Antes de comenzar a trabajar con MySQL, es necesario disponer de un entorno funcional donde ejecutar el servidor. Existen diferentes opciones de instalación según el sistema operativo y las preferencias del usuario.

Opción A: XAMPP (instalación rápida)

XAMPP es un paquete gratuito que integra Apache, MySQL (MariaDB), PHP y Perl. Es la forma más sencilla de montar un entorno de desarrollo local en Windows, Linux o macOS.

  1. Descarga XAMPP desde apachefriends.org.
  2. Instálalo siguiendo los pasos del asistente.
  3. Abre el panel de control y activa los servicios Apache y MySQL.
  4. Comprueba que MySQL funciona accediendo a http://localhost/phpmyadmin.

Opción B: Docker (instalación reproducible)

Si prefieres aislar el entorno, puedes ejecutar MySQL en un contenedor Docker. Crea un archivo llamado docker-compose.yml con el siguiente contenido:

version: "3.9"
services:
  db:
    image: mysql:8.0
    container_name: mysql_daw
    environment:
      - MYSQL_ROOT_PASSWORD=root
      - MYSQL_DATABASE=demodaw
    ports:
      - "3306:3306"
    volumes:
      - ./data:/var/lib/mysql
    restart: unless-stopped

Luego ejecuta en la terminal:

docker compose up -d

Para detener el contenedor:

docker compose down

Verificación

Comprueba el estado con:

docker ps

Y revisa los logs del contenedor con:

docker logs mysql_daw

Actividad práctica: Instala MySQL usando XAMPP o Docker. Aporta una captura del panel de control o del comando SELECT VERSION(); como evidencia.
