MySQL paso a paso: fundamentos, consultas y relaciones

Bienvenido/a a este Recurso Educativo Abierto (REA) diseñado para el módulo de Bases de Datos del ciclo formativo de Desarrollo de Aplicaciones Web (DAW). A lo largo de este recurso aprenderás a instalar, configurar y utilizar el sistema gestor de bases de datos MySQL desde un enfoque teórico–práctico, guiado paso a paso.

Presentación

Este material te permitirá desarrollar las competencias necesarias para administrar y explotar bases de datos relacionales, diseñar esquemas coherentes, aplicar restricciones de integridad y realizar consultas SQL para la extracción y análisis de información.

El REA está estructurado de forma progresiva, combinando contenidos teóricos, ejemplos prácticos, actividades interactivas y proyectos finales que fomentan el aprendizaje autónomo y el razonamiento lógico.

Estructura del REA

  1. Introducción docente y guía del REA: contextualización, objetivos y metodología.
  2. Bienvenida al entorno MySQL: fundamentos de los SGBD y papel de MySQL en el desarrollo web.
  3. Instalación y acceso a MySQL: configuración local, acceso por CLI y GUI, y gestión de usuarios.
  4. Creación y gestión de bases de datos: tipos de datos, creación de tablas, restricciones y normalización.
  5. Consultas básicas en MySQL: sentencias SELECT, filtrado, ordenación, DML y agregaciones.
  6. Relaciones y claves foráneas: diseño relacional, integridad referencial y consultas con JOIN.
  7. Evaluación final y proyecto integrador: aplicación completa de lo aprendido en un caso práctico.
  8. Autoría y Licencia: datos del autor y licencia de uso.

Destinatarios

El REA está destinado al alumnado de segundo curso de Ciclos Formativos de Grado Superior en Informática, especialmente en el módulo de Bases de Datos del ciclo de Desarrollo de Aplicaciones Web. También puede ser útil para docentes o autodidactas interesados en iniciarse en SQL con MySQL.

Metodología

Se aplica un enfoque teórico–práctico y basado en proyectos. Cada bloque combina explicaciones breves con ejercicios guiados que el alumnado puede replicar en su propio entorno. El aprendizaje se refuerza mediante la experimentación, la reflexión y la autoevaluación.

Consejo inicial: Asegúrate de tener un entorno MySQL funcional (XAMPP, Docker o instalación nativa) antes de comenzar. De esta forma podrás poner en práctica los ejemplos y actividades de cada bloque.

Duración total estimada: 12–15 horas · Tipo de contenido: Teórico–práctico · Nivel: Ciclo Formativo de Grado Superior (DAW)
