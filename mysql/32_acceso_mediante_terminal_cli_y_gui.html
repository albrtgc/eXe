3.2 Acceso mediante terminal (CLI) y GUI

Una vez instalado MySQL, existen dos formas principales de interactuar con el servidor: la consola de comandos (CLI) y los entornos gráficos (GUI).

Acceso por terminal (CLI)

Abre una terminal y ejecuta:

mysql -u root -p

Introduce la contraseña y, si la conexión es correcta, verás el prompt de MySQL (mysql>).

Ejecuta una consulta de comprobación:

SHOW DATABASES;
SELECT VERSION();

Acceso mediante entorno gráfico (GUI)

Puedes usar cualquiera de las siguientes herramientas:

  • phpMyAdmin: accesible desde http://localhost/phpmyadmin (incluido en XAMPP).
  • MySQL Workbench: aplicación oficial de MySQL para gestionar y diseñar bases de datos.

En MySQL Workbench, crea una nueva conexión configurando:

  • Host: localhost
  • Puerto: 3306
  • Usuario: root
  • Contraseña: (la que definiste en la instalación)

Prueba de conexión

Ejecuta una consulta simple:

SELECT NOW();

Si devuelve la fecha y hora actuales, la conexión funciona correctamente.

Actividad: Conéctate por CLI y Workbench, ejecuta SELECT VERSION(); y sube capturas de ambas interfaces.
