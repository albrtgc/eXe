5.1. Herramientas principales para el desarrollo

Herramientas Principales para el Desarrollo del Proyecto

En este proyecto utilizarás varias herramientas clave que forman parte del día a día de cualquier desarrollador de software. A continuación se describen las más importantes y cómo se utilizarán en el contexto del proyecto “Repositorio de Biblioteca Escolar”.

En los subapartados encontrarás fichas detalladas sobre:

  • IntelliJ IDEA / Visual Studio Code como entornos de desarrollo (IDE).
  • Git y GitHub para el control de versiones y el trabajo con repositorios.
  • Markdown para la creación de documentación técnica.
  • JUnit y el depurador para probar y mejorar tu código.

Estas herramientas son parte fundamental del desarrollo profesional, por lo que lo que aprendas con ellas en este proyecto te será útil más allá del aula.
